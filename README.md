# ❌⭕ Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. Play with a friend and enjoy a clean, responsive UI.

---

## 🚀 Features

- 👥 Two-player mode (Player X vs Player O)
- 🎯 Detects winner automatically
- 🔄 Restart game functionality
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight (no libraries)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
